# AdminIt

Administrative backend for ruby projects.

[Russian version of this document](README_RU.md)

# Installation

```sh
gem install admin_it
```

or if you using `bundler`:

```ruby
# Gemfile
gem 'admin_it'
```

```sh
bundle install
```
